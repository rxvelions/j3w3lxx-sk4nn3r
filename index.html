♱ 01 Ataque DDoS: Esta opción simula un ataque de denegación de servicio distribuido, enviando múltiples "paquetes" falsos hacia una dirección IP ingresada. Sirve para visualizar cómo se vería un ataque masivo sin causar daño real. ♱ 02 Geolocalización IP: Utiliza un servicio externo para mostrar detalles como país, ciudad, organización, coordenadas GPS y ASN (sistema autónomo) de una dirección IP determinada. Útil para análisis de procedencia y contexto geográfico. ♱ 03 Ver IP Pública: Muestra la IP pública de tu conexión actual. Esta IP es visible para servidores web y puede ser utilizada para saber desde dónde te conectás. ♱ 04 Info del Navegador: Revela el identificador del navegador (User-Agent), el cual incluye sistema operativo, versión del navegador y otros detalles técnicos. Esta información es útil para reconocimiento de cliente. ♱ 05 Encabezados HTTP: Muestra todos los encabezados HTTP que tu navegador envía a los servidores. Son útiles para auditorías y conocer más sobre cómo se identifica tu conexión al hacer peticiones.
